# PWAアイコン

このディレクトリに以下のアイコンを配置してください：

- `icon-192.png` - 192x192ピクセルのPNG画像
- `icon-512.png` - 512x512ピクセルのPNG画像

これらのアイコンはPWAとしてインストールされた際に表示されます。

## アイコンの作成方法

1. オンラインツール（例: [PWA Asset Generator](https://github.com/onderceylan/pwa-asset-generator)）を使用
2. デザインツール（Figma、Photoshop等）で作成
3. 既存のロゴをリサイズ

アイコンは透明背景または白背景で作成することを推奨します。
